## Key directories

**`data/`** holds the Excel workbooks that parameterise each scenario. The `legacy/` subdirectory contains run configuration and default reporting tables inherited from the MESSAGEix-GLOBIOM ENGAGE project and adapted for the Pakistan node.

**`notebooks/`** contains one Jupyter notebook per scenario. Each notebook is self-contained: it builds the scenario from the input workbook, applies scripts, solves with GAMS, and reports results to IAMC format.

**`scripts/`** contains Python modules that apply model modifications shared across scenarios (calibration, constraints, emission bounds). They are imported and called from the notebooks.

**`report/legacy/`** contains the inherited MESSAGEix-GLOBIOM ENGAGE reporting routines. The main entry point is `iamc_report_hackathon.py`, which aggregates and formats model output into IAMC variables.

**`output/`** is the landing directory for timestamped IAMC Excel reports generated by the notebooks. It is tracked by git with a `.gitkeep` placeholder; actual output files are gitignored.
