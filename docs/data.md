# Data

The `data` folder in the MESSAGEix-Canada repository contains all the relevant data required for the preprocess functions. This includes:

- **GDP Data:** Stored in its own folder within the `data` directory, this data is used to adjust demands based on economic activity.
- **Population Data:** Also stored in a separate folder within the `data` directory, this data is used to adjust demands based on demographic changes.

These datasets are critical for ensuring that the model's input data accurately reflects the economic and population conditions across Canada's provinces and territories. The `data` folder is referenced during the preprocessing stage to update parameters and ensure consistency in the model.

Make sure that all necessary data files are correctly placed in the appropriate folders within the `data` directory before running the model.
