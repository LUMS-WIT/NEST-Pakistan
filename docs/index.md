# MESSAGEix-Canada Documentation

Welcome to the official documentation for **MESSAGEix-Canada**, an open energy system integrated model designed at the scale of Canadian provinces and territories, with a five-year time horizon extending until 2060. MESSAGEix-Canada is based on the **MESSAGEix** ([documentation](https://docs.messageix.org/en/latest/)) framework, which formulates energy system planning as a linear programming (LP) problem. The framework's documentation can be found [here](https://docs.messageix.org/), where all core equations are defined and implemented.

## **About MESSAGEix-Canada**

The model was developed using a **rapid prototyping approach**, leveraging the global **MESSAGEix-GLOBIOM** ([documentation](https://docs.messageix.org/projects/global/en/latest/)) framework, specifically the **North America (NAM) region**. The initial step involved downscaling it to a national-scale Canada model, followed by the integration of all **13 provinces and territories**, incorporating suitable assumptions to enhance regional granularity.

## **Prerequisite Skills**
To effectively understand and use MESSAGEix-Canada, familiarity with domain knowledge and scientific computing skills is essential. Users should have a basic understanding of energy system modeling, optimization techniques, and policy analysis. 

For foundational learning, the [Prerequisite](https://docs.messageix.org/en/latest/prereqs.html#prerequisite-knowledge-skills) page of the MESSAGEix documentation provides useful resources to build relevant knowledge.

To gain hands-on experience, it is recommended to start with the [tutorials](https://docs.messageix.org/en/latest/tutorials.html), which offer step-by-step guidance on using the MESSAGEix framework. Additionally, the [teaching resources](https://teaching.ece.iiasa.ac.at/message_ix_workshops.html) include recorded training workshops that can help users familiarize themselves with key concepts and practical implementations.

## **About this Documentation**

This documentation provides a structured guide for users and developers working with MESSAGEix-Canada. It covers the following key aspects:

1. **Installation Instructions** – Steps to set up the required dependencies and environment to run the model.
2. **Running an Existing Scenario** – Guidelines on executing predefined scenarios, interpreting outputs, and troubleshooting common issues.
3. **Model Details** – A comprehensive overview of the model structure, including data sources, repository organization, and methodological framework.
4. **Developing your own scenario** – Instructions for modifying existing scenarios or creating new ones, including parameter adjustments and structural modifications.


## **Project Layout**

```
mkdocs.yml    Configuration file for MkDocs.
docs/         Contains the documentation pages.
data/         Stores external data required by the model, such as GDP and population profiles.
model/        Houses model-related files and scripts.
    notebooks/   Jupyter notebooks for model development and example formulations.
    policies/    Python scripts implementing different policies in the model.
    report/      Contains report files and benchmark data for analysis.
    scenarios/   Includes various scenario configurations and folders:
        baseline/    Business-as-usual (BAU).
        dacs/        Net-zero scenario incorporating direct air capture (DAC) technologies.
        hydrogen/    Scenario exploring different hydrogen technologies.
        netzero/     Basic net-zero scenario.
        netzero_opt/ Optimistic net-zero scenario.
        netzero_pes/ Pessimistic net-zero scenario.
    utilities/   Utilities scripts for model-related tasks.
output/           Stores output Excel files generated by the model.
reporting_outputs/ Contains more detailed Excel reports generated by the model's reporting module.
```
