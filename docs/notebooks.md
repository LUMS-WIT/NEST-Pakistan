# Notebooks

The `notebooks` directory contains Jupyter notebooks that are used to interact with the MESSAGEix-Canada model both before and after solving scenarios.

These notebooks allow users to:

- **Pre-process data:** Modify or inspect input data before running the model.
- **Run scenarios:** Use the `setup_scenario` function to configure and execute the model.
- **Post-process results:** Analyze and visualize the model outputs to gain insights into the energy system scenarios.

These notebooks serve as an interactive platform for users who prefer to work within a Jupyter environment, providing a more hands-on approach to managing and running the model.
